# AIエージェント最小構成

このリポジトリは、AIエージェント運用の「最小限の土台」を提供します。
以下のファイルを起点に、プロジェクト固有のルールや運用フローを追加してください。

## 目的
- エージェントの役割・振る舞いを明文化する
- 永続的に保持すべき知識（長期記憶）を分離する
- タスクの進め方を一貫させる

## ディレクトリ構成
```
.
├─ agent/
│  ├─ system.md      # エージェントの基本設定（人格・制約・目的）
│  └─ behavior.md    # 実行方針・判断基準
├─ memory/
│  └─ long_term.md   # 長期記憶（変更履歴・重要な背景）
├─ tasks/
│  └─ README.md      # タスク運用の簡易テンプレート
└─ README.md
```

## 使い方（最小手順）
1. `agent/system.md` にエージェントの目的と制約を記載
2. `agent/behavior.md` に作業方針・判断の優先順位を記載
3. `memory/long_term.md` に継続的に保持したい情報を追記
4. `tasks/README.md` を元に、タスクテンプレートを必要に応じて拡張
